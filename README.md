# 转发API (AI接口服务中间件)
> 类似`one-api`的项目,但是未作计费,仅做接口转发,支持多种第三方接口转发. 尽最大可能免去频繁适配第三方接口的烦恼.

## 项目初始化

- 安装依赖
```bash
pip install -r requirements.txt
```

- 初始化数据库
`data/init.sql` 文件为数据库初始化文件,请自行导入数据库

## 支持的第三方转发
